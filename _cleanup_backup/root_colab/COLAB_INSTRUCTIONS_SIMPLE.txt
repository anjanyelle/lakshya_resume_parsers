═══════════════════════════════════════════════════════════════════════════
  GOOGLE COLAB TRAINING - SIMPLE 2-CELL METHOD
═══════════════════════════════════════════════════════════════════════════

This method avoids numpy conflicts by using 2 cells with a runtime restart.

═══════════════════════════════════════════════════════════════════════════
  STEP 1: OPEN COLAB & ENABLE GPU
═══════════════════════════════════════════════════════════════════════════

1. Go to: https://colab.research.google.com/
2. Click: Runtime > Change runtime type
3. Select: T4 GPU
4. Click: Save


═══════════════════════════════════════════════════════════════════════════
  STEP 2: CREATE CELL 1 (SETUP)
═══════════════════════════════════════════════════════════════════════════

1. Create a new code cell
2. Open file: COLAB_CELL_1_SETUP.py
3. Copy ALL contents
4. Paste into the cell
5. Run the cell (Shift+Enter)
6. Upload Lakshya-Colab-Training.zip when prompted
7. Wait for setup to complete (~3 minutes)


═══════════════════════════════════════════════════════════════════════════
  STEP 3: RESTART RUNTIME
═══════════════════════════════════════════════════════════════════════════

After Cell 1 completes:

1. Click: Runtime > Restart runtime
2. Click: Yes (confirm)
3. Wait for runtime to restart (~10 seconds)


═══════════════════════════════════════════════════════════════════════════
  STEP 4: CREATE CELL 2 (TRAINING)
═══════════════════════════════════════════════════════════════════════════

1. Create a new code cell
2. Open file: COLAB_CELL_2_TRAIN.py
3. Copy ALL contents
4. Paste into the cell
5. Run the cell (Shift+Enter)
6. Wait for training (~2 hours)
7. Model will auto-download when complete

═══════════════════════════════════════════════════════════════════════════

Cell 1 (Setup):          ~3 minutes
Runtime Restart:         ~10 seconds

Download:                ~1 minute

TOTAL:                   ~2 hours 5 minutes

═══════════════════════════════════════════════════════════════════════════
  WHY 2 CELLS?
═══════════════════════════════════════════════════════════════════════════

Colab has pre-installed packages that conflict with numpy 1.26.4.
By restarting the runtime after installation, we clear these conflicts.
This ensures training runs smoothly without errors.

  FILES YOU NEED
═══════════════════════════════════════════════════════════════════════════

✅ Lakshya-Colab-Training.zip (964 KB) - Upload to Colab
✅ COLAB_CELL_1_SETUP.py - Paste in Cell 1
✅ COLAB_CELL_2_TRAIN.py - Paste in Cell 2
