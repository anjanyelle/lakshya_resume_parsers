═══════════════════════════════════════════════════════════════════════════
  HOW TO TRAIN YOUR MODEL ON GOOGLE COLAB
═══════════════════════════════════════════════════════════════════════════

📦 YOU HAVE 2 FILES:

1. Lakshya-Colab-Training.zip  (388 KB) - Training data package
2. PASTE_IN_COLAB.py           - Complete training script


═══════════════════════════════════════════════════════════════════════════
  STEP-BY-STEP INSTRUCTIONS
═══════════════════════════════════════════════════════════════════════════

STEP 1: OPEN GOOGLE COLAB
──────────────────────────────────────────────────────────────────────────
Go to: https://colab.research.google.com/


STEP 2: ENABLE GPU
──────────────────────────────────────────────────────────────────────────
1. Click: Runtime > Change runtime type
2. Select: T4 GPU
3. Click: Save


STEP 3: CREATE NEW CELL
──────────────────────────────────────────────────────────────────────────
Click: + Code (top left)


STEP 4: PASTE THE SCRIPT
──────────────────────────────────────────────────────────────────────────
1. Open: PASTE_IN_COLAB.py (on your computer)
2. Copy: ALL the contents (Ctrl+A, Ctrl+C)
3. Paste: Into the Colab cell (Ctrl+V)


STEP 5: RUN THE CELL
──────────────────────────────────────────────────────────────────────────
Click: Play button (▶) or press Shift+Enter


STEP 6: UPLOAD ZIP WHEN PROMPTED
──────────────────────────────────────────────────────────────────────────
When you see "Choose Files" button:
1. Click it
2. Select: Lakshya-Colab-Training.zip
3. Wait for upload to complete


STEP 7: WAIT FOR TRAINING
──────────────────────────────────────────────────────────────────────────
⏱️  Training takes ~60-70 minutes
📊 You'll see progress updates every 100 steps
🎯 Final F1 Score should be ~96%


STEP 8: DOWNLOAD MODEL
──────────────────────────────────────────────────────────────────────────
When training completes:
✅ Model will automatically download as: resume-ner-deberta.zip
📥 Check your browser's Downloads folder


═══════════════════════════════════════════════════════════════════════════
  WHAT YOU'LL SEE
═══════════════════════════════════════════════════════════════════════════

The script will show:
✅ STEP 1: VERIFY GPU
✅ STEP 2: UPLOAD ZIP FILE
✅ STEP 3: EXTRACT ZIP
✅ STEP 4: NAVIGATE TO AI-SERVICE
✅ STEP 5: INSTALL DEPENDENCIES
✅ STEP 6: START TRAINING (~60-70 minutes)
✅ STEP 7: VERIFY TRAINING RESULTS
✅ STEP 8: DOWNLOAD MODEL


═══════════════════════════════════════════════════════════════════════════
  AFTER TRAINING
═══════════════════════════════════════════════════════════════════════════

1. Extract resume-ner-deberta.zip
2. Copy the folder to your project:
   ai-service/models/resume-ner-deberta/
3. Restart your AI service
4. Test with real resumes!


═══════════════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

❌ "No GPU detected"
   → Runtime > Change runtime type > T4 GPU

❌ "File not found"
   → Make sure you uploaded Lakshya-Colab-Training.zip

❌ "Training failed"
   → Check the error message in the output
   → Make sure you have enough GPU memory

❌ Dependency warnings about numpy
   → These are NORMAL - ignore them!
   → Training will work fine
